Open Compliance Hub – UAE IAR Compliance Portal




Lightweight • Open Source • Audit-Ready • Executive Dashboards

Overview
Open Compliance Hub is a lightweight, open-source compliance management platform designed to help organizations assess, track, and report their implementation of UAE Information Assurance Regulation (UAE IAR v1.1) controls.
The platform provides real-time executive visibility, risk-based implementation tracking, session-based working, versioned backups, and audit-ready reporting, making it ideal for organizations preparing for certification or building continuous compliance programs.

Key Value
	• Real-time compliance posture visibility
	• Executive dashboards for leadership
	• Risk and priority-driven control tracking
	• Centralized UAE IAR control management
	• Statement of Applicability (SoA) generation
	• Monthly / yearly version snapshots
	• JSON backup & restore
	• Zero-cost open-source alternative to GRC tools

Core Features
Executive Dashboard
	• Overall compliance percentage
	• Domain-wise maturity view
	• Implementation status distribution
	• Documentation coverage
	• Real-time posture monitoring

Control Management
	• Complete UAE IAR v1.1 control library
	• Status tracking:
		○ Fully Implemented
		○ Partially Implemented
		○ Planned
		○ Not Implemented
		○ Not Applicable
	• Bulk updates and filtering

Risk & Priority Tracking
	• Applicability classification:
		○ Always Applicable
		○ Risk-Based
	• Priority levels:
		○ P1 – Critical
		○ P2 – High
		○ P3 – Medium
		○ P4 – Low
	• Critical control progress tracking

Reporting & Audit
	• Statement of Applicability (SoA)
	• Full control export
	• PDF and Excel formats
	• Audit-ready structure for assessments and certification

Data Management & Version Control
	• Session-based working environment
	• JSON backup and restore
	• Monthly / yearly compliance snapshots
	• Historical version tracking
	• Framework reset and initialization
	• Full data portability
